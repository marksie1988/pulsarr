# Help Command

The help command displays all of the Cogs (modules) for this bot

To see the modules:

```shell
/pulsarr help
```

To see the commands for the specific cog:

```shell
/pulsarr help <cog>
```

```shell
/pulsarr help arrCommands
/pulsarr help Settings
```
